Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM:

1. document.getElementById("id-name") => Selects an element by its id.

2. document.getElementsByClassName("class-name") => Selects all elements with a specific class.

3. document.getElementsByTagName("tag-name") => Selects all elements with a specific tag.

4. document.querySelector("css-selector") => Selects the first element that matches the given CSS selector.

5. document.querySelectorAll("css-selector") => Selects all elements that match the given CSS selector and returns a NodeList.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML, innerText, and textContent are:

  1. innerHTML
    • Returns or sets the HTML content inside an element.
    • Recognizes and renders HTML tags.
  2. innerText
    • Returns or sets the visible text inside an element.
    • Ignores hidden text.
  3. textContent
    • Returns or sets all text content inside an element, including hidden text.
    • Does not recognize HTML tags.

Question-3: What is event delegation in the DOM?

Event Delegation in the DOM:

Event Delegation allows a parent element to handle events for multiple child elements using event bubbling. Instead of adding listeners to each child, we add one to the parent and use event.target to detect which child triggered the event.

Question-4: What is event bubbling in the DOM?

Event Bubbling in the DOM:

Event Bubbling is a phase in the event propagation process where an event starts from the target element and moves upward to its parent, grandparent, and so on, until it reaches the document object.

Creating, Adding, and Removing Elements in JavaScript:
